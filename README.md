# .github

Stewardship-focused fork of the Ranvier MUD engine, keeping it buildable and usable on modern platforms.

## Forking RanvierMUD Repositories

This repository contains documentation and tools for forking all RanvierMUD repositories into the Ranvire organization.

See [FORKING_INSTRUCTIONS.md](./FORKING_INSTRUCTIONS.md) for detailed instructions on forking all 30 repositories from the RanvierMUD organization while preserving commit history, branches, and tags.

### Quick Start

```bash
# Using the provided script (requires GitHub CLI)
./fork-repositories.sh
```

Or fork repositories manually using the GitHub web interface (see full documentation for details).
